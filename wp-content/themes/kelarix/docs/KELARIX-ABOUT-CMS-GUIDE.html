1Page Setup

Publish a WordPress Page that uses the About Us template.

Create the page.
Dashboard → Pages → Add New. Title it About Us. Slug: about.
Assign the template.
In the right sidebar → Page Attributes → Template → choose About Us.
Publish.
Click Publish. Preview at /about.
Add to menu (optional).
Appearance → Menus → Primary Menu, add the About page.
The theme file backing this is page-about.php. It automatically pulls in the 6 sections under template-parts/about/. Every field below shows design defaults until you fill it in — the page is never blank.

2ACF Field Group — “About Page Content”

Create one field group in ACF → Field Groups → Add New.
Title: About Page Content  ·  Location: Page is equal to About Us (or the About page you created above).
Add a Tab field at the start of each section to keep the editor organised.

3Reusable CPT — Process Steps

The About page's "structured path" accordion reuses the same kx_process posts as the homepage. No duplicate content required.

If you already created the 6 Process Step posts for the homepage, the About accordion will populate automatically.
If not, follow the same setup as the homepage guide (§3, "Process Steps").
OrderTitleDescription
1DiagnoseUnderstand the business problem, visibility gaps, workflow pain, and decision needs.
2MapMap processes, data sources, roles, systems, bottlenecks, and reporting gaps.
3DesignDefine the right system, workflows, dashboards, user journeys, architecture, and success criteria.
4BuildDevelop the data layer, dashboards, automation, AI assisted features, and custom software.
5IntegrateConnect the system with existing tools, data sources, workflows, and teams.
6ImproveRefine the system based on usage, feedback, performance, and changing business needs.
