1Page Setup

Publish a WordPress Page that uses the Industries template.

Create the page.
Dashboard → Pages → Add New. Title it Industries. Slug: industries.
Assign the template.
Right sidebar → Page Attributes → Template → choose Industries.
Publish.
Click Publish. Preview at /industries.
Add to menu.
Appearance → Menus → Primary Menu, add the Industries page.
Theme file: page-industries.php. Section parts live under template-parts/industries/. All defaults render even without ACF filled — page never blank.
